# 課題まとめ（モック）

ローカルで動かせる簡易UIです。`mock/tasks.json` を読み込み、一覧表示・フィルタ・検索ができます。

起動方法（推奨：VS Code の Live Server）:

PowerShell の簡単な例:

```powershell
cd C:\Users\chiyo\lecture\pis3\hackathon-NCP
# python があれば
python -m http.server 5500
# もしくは node があれば
npx serve . -p 5500
```

ブラウザで `http://localhost:5500/index.html` を開いてください。
